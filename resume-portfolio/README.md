# Resume Portfolio

This project is a personal resume portfolio that showcases my professional experience, education, skills, and includes a link to my GitHub profile.

## Project Structure

```
resume-portfolio
├── index.html        # Main HTML document for the resume
├── css
│   └── styles.css    # Styles for the resume
├── js
│   └── main.js       # JavaScript functionality for the resume
├── assets
│   └── fonts         # Directory containing font files
└── README.md         # Documentation for the project
```

## How to View the Resume

1. Clone the repository to your local machine.
2. Open the `index.html` file in your web browser to view the resume.

- HTML

- JavaScript
